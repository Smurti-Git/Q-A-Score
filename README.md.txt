# Subjective Answer Scorer

A simple web-based tool to evaluate subjective answers based on keyword similarity. This tool is ideal for trainers, educators, and internal reviewers to score written responses using a quick matching technique.

## 🚀 Features

- Enter a question, correct answer, and a user's answer.
- Instantly get a similarity percentage score.
- Helps provide quick feedback on subjective responses.
- Easy to use and lightweight – runs entirely in the browser.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📦 Files

- `index.html`: Main HTML file for the user interface.
- `script.js`: Contains logic to evaluate and score user answers.

## 🔧 How to Use

1. Open `index.html` in your browser.
2. You will see three text areas:
   - **Question**: Type the question you're evaluating.
   - **Correct Answer**: Enter the ideal answer.
   - **User's Answer**: Paste or write the user's answer.
3. Click **Evaluate Answer** to view the score percentage based on keyword match.

## 🧭 Answer Format (Important Guideline)

To ensure proper evaluation and consistency in responses, users **must** write answers in a **navigation-style** format. This means the answer should **begin with the question itself**, followed by a clear step-by-step instruction or explanation.

### Example:

**Question**:  
How to set same unit for multiple items at once?

**Correct Format for Answer**:
 Using assign unit : click on item - 3 dot, next Tu add item - assign unit - select items - next - select unit and save .
